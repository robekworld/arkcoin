# arkcoin

Dashboard for arkcoin.net

## Installation

```
cd arkcoin
npm install
npm install --save react-router-dom
```

## Run for developement

```
npm start
```

## Build for production

```
git clone https://github.com/Jarunik/arkcoin
cd arkcoin
npm install
npm install --save react-router-dom
npm run build
mv build build_final
```
